## Formato de cada vídeo (4 telas)

1. **Hook + foto do produto + preço** (com compare riscado)
2. **Foto grande** + frete grátis / devolução
3. **Why Trove?** — 4 vantagens de comprar conosco
4. **CTA** trove-us.com

## Como gerar os vídeos

```bash
npm run tiktok:ads:batch2    # 16 novos (16–31)
npm run tiktok:ads             # todos os 31
npm run tiktok:ads:pet         # 5 pet
npm run tiktok:ads:wellness    # 5 wellness
npm run tiktok:ads:tech        # 5 tech
npm run tiktok:ads:home        # 3 home
```

## Como postar no TikTok (EUA)

1. **Conta**: TikTok Business ou creator normal
2. **Link na bio**: `https://trove-us.com` (ou link direto do produto)
3. **Pixel/Meta**: se for rodar ads pagos depois, use o mesmo criativo no TikTok Ads Manager
4. **Horários teste**: 7–9h, 12–14h, 19–22h (horário local EUA — EST/PST)
5. **Som**: no TikTok app, troque por trending sound **depois** de uploadar (vídeo sem som ou com som neutro)

## Dica de estratégia (“ganhar sempre”)

- 1 vídeo por dia > 1 vídeo viral e parar
- Teste 3 hooks diferentes no mesmo produto
- Comece **sem ads pagos** — orgânico 7 dias, veja qual hook retém
- Só depois coloque $5–10/dia no que já teve >3s retenção
